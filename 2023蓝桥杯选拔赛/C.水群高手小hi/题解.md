## 水群高手小 hi 题解

数字位数少于 $len(n)$ 的数必然小于 $n$，此时的答案一共有 $9(len(n)-1)+1$ 个，位数等于 $len(n)$ 时，至少有 $x-1$ 个自然数满足答案，其中 $x$ 为 $n$ 的最高位，最后特判 $len(n)$ 位 $x$ 是否满足题意即可。

```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
    string str, ss;
    cin >> str;
    for (char ch: str) {
        ss += str[0];
    }
    int ans = (str.length() - 1) * 9 + 1 + (str[0] - '0');
    if (str >= ss) {
        cout << ans << endl;
    } else {
        cout << ans - 1 << endl;
    }

    return 0;
}
```
